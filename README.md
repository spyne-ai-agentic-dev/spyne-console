# Spyne Console

A basic Node + Express web application.

## Getting started

```bash
npm install
npm start
```

Then open http://localhost:3000

## Project structure

- `server.js` — Express server and `/api/health` endpoint
- `public/` — static front-end (HTML, CSS, JS)

## Scripts

- `npm start` — run the server
- `npm run dev` — run with auto-reload (`node --watch`)
