# Shopify Design System

A cinematic commerce platform running two parallel tracks. Marketing-hero/narrative pages live on near-black canvases with full-bleed merchant photography and giant thin Neue Haas Grotesk display; transactional pages flip to a cream-mint canvas with pastel greens. The polarity flip is the brand.

## Core tokens (cheat-sheet)
- **Primary / ink**: `#000000`; on-dark `#ffffff`
- **Canvases**: night `#000000` / elevated `#0a0a0a`; light `#ffffff`; cream `#fbfbf5`; dark surface `#1e2c31`
- **Shades**: `#d4d4d8` → `#a1a1aa` → `#71717a` → `#52525b` → `#3f3f46`
- **Hairlines**: light `#e4e4e7`; dark `#1e2c31`
- **Pastel greens (transactional)**: aloe `#c1fbd4`, pistachio `#d4f9e0`; cool links `#9dabad` / `#9797a2` / mint `#99b3ad`
- **Type**: Neue Haas Grotesk display at thin weights (marketing); Inter for UI body (transactional)
- **Buttons**: black pill stroked white on dark; same pill vocabulary on light

## How to apply
1. Read `reference/DESIGN.md` for the full token set, typography, and component specs across both tracks.
2. Pick the track by page type: dark cinematic for marketing/narrative, cream-mint for pricing/signup/dashboards.
3. Keep the pill-CTA vocabulary shared across both; let canvas polarity carry the brand.

The complete specification lives in `reference/DESIGN.md`.
