# Lovable Design System

Warmth through restraint: a creamy parchment canvas that separates it from cold-white dev-tool sites, near-black text, and a humanist display typeface that carries the brand personality.

## Core tokens (cheat-sheet)
- **Canvas (parchment)**: `#f7f4ed`; surfaces `#eceae4` / `#fcfbf8` / white `#ffffff`
- **Ink**: `#1c1c1c` / muted `#5f5f5d`
- **Accent**: blue `#3b82f6` — used very sparingly
- **Type**: "Camera Plain" — humanist warm sans with rounded terminals and editorial letter-spacing at display; fallback `Inter, system-ui, sans-serif`
- **Feel**: approachable, analog, "well-crafted notebook"; minimal ornament

## How to apply
1. Read `reference/DESIGN.md` for the full atmosphere, type, color, and component notes (this spec is prose-formatted, not token frontmatter).
2. Keep the parchment canvas central — it's the core brand move against cold-white norms.
3. Let the humanist display type carry personality; use accent blue only for the rare highlight.

The complete specification lives in `reference/DESIGN.md`.
