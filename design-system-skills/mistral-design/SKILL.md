# Mistral AI Design System

A warm AI brand with a singular signature: atmospheric sunset gradients (mustard → orange → deep red) over mountain photography, plus a horizontal "sunset stripe" closing band on every page.

## Core tokens (cheat-sheet)
- **Primary (orange)**: `#fa520f` (deep `#cc3a05`)
- **Sunshine ramp**: `#ffd06a` → `#ffb83e` → `#ffa110` → `#ff8105`; saturated yellow `#ffd900`
- **Canvas (cream)**: `#fff8e0` / light `#fffaeb` / deeper `#fff0c2`; beige `#e6d5a8`
- **Ink**: `#1f1f1f` / tint `#3d3d3d` / charcoal `#2c2c2c` / steel `#6a6a6a`
- **Type**: elegant near-serif for hero displays; see reference for body stack
- **Signature**: sunset gradient bands + "sunset stripe" bar closing each page

## How to apply
1. Read `reference/DESIGN.md` for the full token set, typography, components, and gradient/stripe specs.
2. Keep cream surfaces warm; reserve the orange CTA as the saturated voltage against soft sunshine tones.
3. Close pages with the signature sunset stripe.

The complete specification lives in `reference/DESIGN.md`.
