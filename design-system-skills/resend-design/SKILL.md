# Resend Design System

A near-pure-black canvas with off-white text and a deep editorial-serif (Domaine) signature, punctuated by glowing functional accents.

## Core tokens (cheat-sheet)
- **Primary / ink (off-white)**: `#fcfdff` (on it: `#000000`); body `rgba(252,253,255,0.86)`
- **Canvas**: `#000000`; cards `#0a0a0c`; elevated `#101012`; deep `#06060a`
- **Muted**: `#a1a4a5` / ash `#888e90` / stone `#464a4d`
- **Hairlines**: `rgba(255,255,255,0.06)` / strong `rgba(255,255,255,0.14)`
- **Glowing accents**: orange `#ff801f`, yellow `#ffc53d`, blue `#3b9eff`, green `#11ff99` (with glow halos)
- **Type**: editorial serif Domaine as the signature display accent; see reference for body stack

## How to apply
1. Read `reference/DESIGN.md` for the full token set, typography, and component specs.
2. Keep the canvas near-pure black with off-white text; use accents as glowing, sparing highlights.
3. Reserve the Domaine serif for the editorial display moments.

The complete specification lives in `reference/DESIGN.md`.
