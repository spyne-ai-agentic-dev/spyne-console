# Zapier Design System

A workflow-automation platform with a confident-warm voice: cream neutrals, deep coffee ink, and a single saturated orange CTA accent.

## Core tokens (cheat-sheet)
- **Primary (orange)**: `#ff4f00` (on-primary `#fffefb`)
- **Canvas (cream)**: `#fffefb` / soft `#f8f4f0`
- **Ink (coffee)**: `#201515` / soft `#2f2a26` / mid `#36342e`
- **Body**: `#605d52` / mid `#939084` / mute `#c5c0b1`
- **Type**: Degular Display at hero scale; Inter for sub-displays and body

## How to apply
1. Read `reference/DESIGN.md` for the full token set, typography, and component specs.
2. Keep neutrals warm (cream + coffee ink); use the orange CTA as the single saturated accent.
3. Pair Degular Display heroes with Inter body for the confident-warm voice.

The complete specification lives in `reference/DESIGN.md`.
