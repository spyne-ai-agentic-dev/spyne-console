# Vercel (Geist) Design System

A stark black-on-near-white developer-platform aesthetic. A single ink tone carries every heading, CTA, and border; the only color allowed is a multi-stop mesh gradient confined to the hero.

## Core tokens (cheat-sheet)
- **Primary / ink**: `#171717` — headings, CTAs, borders all in one tone
- **Canvas**: `#fafafa`; elevated `#ffffff`; body `#4d4d4d`; mute `#8f8f8f`
- **Hairlines**: `#ebebeb` / `#f2f2f2`
- **Link**: `#0070f3` (deep `#0761d1`)
- **Semantic**: error `#ee0000`, warning `#f5a623`
- **Hero mesh gradient (hero ONLY)**: violet `#7928ca`, cyan `#50e3c2`, pink `#ff0080` (+ blue/amber)
- **Type**: Geist Sans for tightly-tracked display; Geist Mono for technical eyebrow labels
- **Radius**: pill-shaped black buttons for marketing CTAs; 6px square buttons for app/nav chrome

## How to apply
1. Read `reference/DESIGN.md` for the full token set, typography, and component specs.
2. Keep the palette monochrome; confine the mesh gradient strictly to the hero.
3. Use Geist Mono for eyebrows/labels to signal the technical voice.

The complete specification lives in `reference/DESIGN.md`.
