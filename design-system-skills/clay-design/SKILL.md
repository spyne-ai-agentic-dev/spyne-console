# Clay Design System

A vibrant claymation-meets-data interface: warm cream canvas, dark-navy CTAs, rounded display type, and saturated single-color feature cards. Brand voltage comes from 3D-rendered claymation illustrations and bright multi-color card surfaces.

## When to use
Apply when building or restyling any GTM, data-orchestration, or playful B2B SaaS interface or long-scroll explainer page that should feel like Clay.

## Core tokens (cheat-sheet)
- **Primary**: near-black `#0a0a0a` (active `#1f1f1f`) — dark CTAs
- **Canvas**: warm cream `#fffaf0`; surfaces soft `#faf5e8`, card `#f5f0e0`, strong `#ebe6d6`; dark `#0a1a1a` / elevated `#1a2a2a`
- **Ink**: `#0a0a0a` / body `#3a3a3a` / muted `#6a6a6a`
- **Hairlines**: `#e5e5e5` / `#f0f0f0`
- **Brand feature-card palette**: pink `#ff4d8b`, teal `#1a3a3a`, lavender `#b8a4ed`, peach `#ffb084`, ochre `#e8b94a`, mint `#a4d4c5`, coral `#ff6b5a`
- **Semantic**: success `#22c55e`, warning `#f59e0b`, error `#ef4444`
- **Type**: custom rounded display face (see reference for stack/scale and substitutes)

## How to apply
1. Read `reference/DESIGN.md` for the full token set, typography scale, component specs, and responsive rules.
2. Keep the cream canvas warm; use saturated single-color cards to punctuate long-scroll pages.
3. Lead heroes with 3D claymation illustration; CTAs stay dark-navy/near-black.

The complete specification lives in `reference/DESIGN.md`.
