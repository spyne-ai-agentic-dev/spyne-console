# ElevenLabs Design System

A voice-AI brand whose marketing reads like an editorial print magazine: off-white canvas, warm near-black ink, and pastel atmospheric gradient orbs as the only "color". Brand voltage is photographic, not chromatic.

## Core tokens (cheat-sheet)
- **Primary (ink pill)**: `#292524` (active `#0c0a09`); secondary is a transparent outline
- **Canvas**: off-white `#f5f5f5` / soft `#fafafa`; cards `#ffffff`; dark `#0c0a09` / elevated `#1c1917`
- **Ink**: `#0c0a09` / body `#4e4e4e` / muted `#777169` / soft `#a8a29e`
- **Hairlines**: `#e7e5e4` / `#f0efed` / strong `#d6d3d1`
- **Gradient orbs (decorative)**: mint `#a7e5d3`, peach `#f4c5a8`, lavender `#c8b8e0` (+ sky) — drifting atmospheric only
- **Type**: Waldenburg Light weight 300 for display (editorial signature); Inter for body/nav/captions

## How to apply
1. Read `reference/DESIGN.md` for the full token set, typography, and component specs.
2. No neon accent, no saturated CTA, no dark dev-tools canvas — modest weights and atmospheric photography do the work.
3. Keep gradient orbs soft and decorative; never use them on CTAs or structure.

The complete specification lives in `reference/DESIGN.md`.
